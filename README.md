## Features
- *Smooth* animations (thanks failed)
- Clean but functional bar
- Bspwm config with all the basic fixes you need.
- Aesthetic dashboard

## Featuring
- **Distro**     -> Fedora 41
- **WM**         -> bspwm
- **Shell**      -> zsh
- **Terminal**   -> kitty
- **Compositor** -> picom
- **Widgets**    -> eww
